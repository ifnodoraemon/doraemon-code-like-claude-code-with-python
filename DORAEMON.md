# Doraemon Code Project Rules

Project specific rules for Doraemon Code.

## Tech Stack
- Language: Python 3.10+
- Framework: FastAPI

## Code Style
- 4 space indentation
- Type hints required

## Directory Structure
- src/: Source code
- tests/: Tests

## Environment Variables
- `DORAEMON_MODEL`: Override default model (default: gemini-3-pro-preview)
