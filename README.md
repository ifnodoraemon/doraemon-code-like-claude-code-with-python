# 🤖 Doraemon Code

> **"从口袋里掏出任何你需要的工具"**

一个**开源的、本地运行的 Claude Code 替代品**，支持 Google/OpenAI/Anthropic/Ollama 多种模型，用 3 个统一工具替代 15+ 分散工具。

[![Python](https://img.shields.io/badge/python-3.10%2B-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
[![Tools](https://img.shields.io/badge/tools-39-green)](src/host/tools.py)

## ⚡ 30 秒快速开始

```bash
git clone https://github.com/ifnodoraemon/doraemon-code.git
cd doraemon-code && pip install -e .

export GOOGLE_API_KEY=your_key   # 或 OPENAI_API_KEY / ANTHROPIC_API_KEY

doraemon
```

## 🔄 数据流详解

### 1. 用户输入处理流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           用户输入处理流程                                   │
└─────────────────────────────────────────────────────────────────────────────┘

用户输入: "帮我看看 @./src/main.py 有什么问题"
    │
    ▼
┌─────────────────┐
│ 1. @文件展开    │  @./src/main.py ──▶ 读取文件内容 ──▶ 嵌入到输入中
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 2. 命令检测     │  以 / 开头? ──▶ 路由到命令处理器
└────────┬────────┘  以 ! 开头? ──▶ 直接执行 shell
         │
         ▼
┌─────────────────┐
│ 3. 上下文管理   │  添加到历史 ──▶ 检查 token 限制 ──▶ 必要时摘要压缩
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 4. 构建 Prompt  │  System Prompt + Rules + Memory + Skills + History
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 5. 调用 LLM     │  流式输出 ──▶ 实时显示 ──▶ 收集工具调用
└─────────────────┘
```

### 2. Agentic 工具循环

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          Agentic 工具循环                                    │
└─────────────────────────────────────────────────────────────────────────────┘

                    ┌──────────────────────────────────┐
                    │         LLM 响应                 │
                    │  "我需要先读取文件..."           │
                    │  tool_calls: [read(...)]         │
                    └───────────────┬──────────────────┘
                                    │
                                    ▼
                    ┌──────────────────────────────────┐
                    │       有工具调用?                │
                    └───────────────┬──────────────────┘
                           是 │            │ 否
                              ▼            ▼
              ┌───────────────────┐    ┌──────────────┐
              │   权限检查        │    │  输出结果    │
              │   ALLOW/ASK/DENY  │    │  结束循环    │
              └─────────┬─────────┘    └──────────────┘
                        │
           ┌────────────┼────────────┐
           ▼            ▼            ▼
      ┌────────┐   ┌────────┐   ┌────────┐
      │ ALLOW  │   │  ASK   │   │ DENY   │
      │ 直接   │   │ 用户   │   │ 拒绝   │
      │ 执行   │   │ 确认   │   │ 返回   │
      └───┬────┘   └───┬────┘   └────────┘
          │            │
          └─────┬──────┘
                ▼
    ┌──────────────────────────────────┐
    │        并行执行工具              │
    │  ┌──────┐ ┌──────┐ ┌──────┐     │
    │  │read  │ │search│ │ ...  │     │
    │  └──┬───┘ └──┬───┘ └──┬───┘     │
    │     └────────┼────────┘         │
    │              ▼                  │
    │     asyncio.gather()            │
    └───────────────┬─────────────────┘
                    │
                    ▼
    ┌──────────────────────────────────┐
    │      结果回传给 LLM              │
    │  messages += tool_results        │
    │  response = await chat(...)      │
    └───────────────┬──────────────────┘
                    │
                    └──────▶ 循环继续 (最多 15 轮)
```

### 3. 上下文管理与压缩

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         上下文管理与压缩                                     │
└─────────────────────────────────────────────────────────────────────────────┘

                    消息历史
    ┌─────────────────────────────────────────┐
    │ [user] 第1轮问题                        │
    │ [assistant] 第1轮回答                   │
    │ [user] 第2轮问题                        │  ◀── 旧消息
    │ [assistant] 第2轮回答 + tool_calls      │      (将被摘要)
    │ [tool] 工具结果                         │
    │ [assistant] 继续回答                    │
    ├─────────────────────────────────────────┤
    │ [user] 第N轮问题                        │  ◀── 保留最近6条
    │ [assistant] 第N轮回答                   │      (3轮对话)
    │ [user] 当前问题                         │
    └─────────────────────────────────────────┘

    当 token 使用率 > 70%:
    ┌─────────────────────────────────────────┐
    │ [summary] 之前的对话摘要:               │  ◀── LLM 生成摘要
    │   - 讨论了 X 功能的实现                 │
    │   - 修改了 Y 文件                       │
    │   - 遇到了 Z 问题并解决                 │
    ├─────────────────────────────────────────┤
    │ [user] 第N轮问题                        │  ◀── 保留最近消息
    │ [assistant] 第N轮回答                   │
    │ [user] 当前问题                         │
    └─────────────────────────────────────────┘

    关键: tool_call 和 tool_result 成对保留，不会被拆散
```

## 📋 命令系统

### CLI 参数

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                            CLI 参数一览                                      │
└─────────────────────────────────────────────────────────────────────────────┘

doraemon [OPTIONS]

会话管理:
  --resume, -r [ID]      恢复会话 (无ID则显示选择器)
  --continue, -c         继续最近的会话
  --project, -p NAME     项目隔离
  --name, -n NAME        新会话名称

运行模式:
  --print                非交互模式 (执行后退出)
  --max-turns N          限制对话轮次
  --verbose, -v          详细日志

工具控制:
  --allowedTools LIST    允许的工具 (逗号分隔)
  --disallowedTools LIST 禁止的工具 (逗号分隔)

示例:
  doraemon                           # 交互模式
  doraemon -c                        # 继续上次
  doraemon --resume                  # 选择历史会话
  doraemon --resume abc123           # 恢复指定会话
  doraemon -p "query" --print        # 非交互执行
  doraemon --project MyApp           # 项目隔离
  doraemon --max-turns 5             # 限制5轮
```
